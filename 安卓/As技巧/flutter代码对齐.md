在Android Studio和IntelliJ中自动格式化代码
安装Dart插件（请参阅编辑器设置），以便在Android Studio和IntelliJ中自动格式化代码。

### as中选中代码后右键，Reformat code with dartfmt

要在当前源代码窗口中自动格式化代码，请右键单击代码窗口并选择Reformat code with dartfmt。您也可以通过快捷键来格式化代码。
